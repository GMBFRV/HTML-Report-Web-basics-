HTML-таблиця

Основні теги для створення таблиці в HTML включають наступні:

  1. '<table>': Визначає саму таблицю.
  2. '<tr>': Визначає рядок у таблиці.
  3. '<th>': Визначає заголовок стовпця. Зазвичай використовується для першого рядка або першого стовпця таблиці.
  4. '<td>': Визначає окрему комірку в таблиці.

Приклад коду таблиці:

    
                <table>
                    <tr>
                        <td>Ячійка 1</td>
                        <td>Ячійка 2</td>
                    </tr>
                    <tr>
                        <td>Ячійка 3</td>
                        <td>Ячійка 4</td>
                    </tr>
                </table>

Цей код виглядатиме на сторінці ось так:

Cell 1 Cell 2
Cell 3 Cell 4

HTML-форма

Основні теги для створення форми в HTML включають наступні:

  1. '<form>': Визначає форму і вказує на маршрут, до якого будуть надіслані дані після натискання кнопки "Надіслати".
  2. '<label>': Визначає мітку для керуючого елемента форми.
  3. '<input>': Визначає поля для введення тексту, адреси електронної пошти та радіо-кнопки.
  4. '<textarea>': Визначає поле для введення багаторядкового тексту.
  5. '<submit>': Визначає кнопку для надсилання даних форми.

Приклад HTML-коду форми:

    
            <form>
                <label for="name">Ім'я:</label><br>
                <input type="text" id="name" name="name"><br>

                <label for="email">Електронна пошта:</label><br>
                <input type="email" id="email" name="email"><br>

                <label for="message">Повідомлення:</label><br>
                <textarea id="message" name="message" rows="4" cols="50"></textarea><br>

                <label>Стать:</label><br>
                <input type="radio" id="male" name="gender" value="male">
                <label for="male">Чоловіча</label><br>
                <input type="radio" id="female" name="gender" value="female">
                <label for="female">Жіноча</label><br>

                <input type="submit" value="Надіслати">
            </form>
