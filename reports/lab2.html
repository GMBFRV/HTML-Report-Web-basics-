Тема: Каскадні таблиці стилів. Селектори. Ідентифікатори. Стильове оформлення текстових елементів в HTML-документах.

Мета: придбати практичні навички роботи з селекторами, ідентифікаторами, списками, різноманітними властивостями кольору і фону, зовнішними та внутрішними відступами, плаваючими елементами, оформленням текстових елементів.

Способи підключення стилів

  1. Вбудовані стилі: 
    
    <h3 style="font-weight: bold; margin: 0.5em 0">Кошик</h3>
  2. Зовнішні стилі: 
    
    <link rel="stylesheet" href="css/style.css">

    У файлі 'styles.css':
    header {
            display: flex;
            flex-direction: row;
            align-items: center;
            justify-content: space-around;
    }

Селектори тегу:

Приклад використання:

HTML код: <header>...</header>
CSS код: header { background-color: #333333; padding: 20px; width: 100%; }

Селектори класу:

Приклад використання:

HTML код: <a href="#about" class="nav-link">ПРО НАС</a>
CSS код: .nav-link { color: white; padding: 10px; }

Селектори ідентифікаторів:

Приклад використання:

HTML код: <div id="promotion"></div>
CSS код: #promotion { text-align: center; }

Інші селектори:

  • Дочірні селектори:
    HTML код: <div id="promotion"> <img src="images/2.jpg" alt="Promotion Image"> </div>
    CSS код: #promotion img { margin-top: 50px; width: 50%; height: auto; }

    • CSS: Шрифти Текст Таблиці Фон Контур Списки СSS Просунутий

    1. Шрифти:
      body { font-family: 'Merriweather', serif; }
    2. Фон:
      body { background-color: antiquewhite; }
    3. Колір тексту:
      .nav-link { color: white; padding: 10px; }

    Висновок з лабораторної роботи

    Для проекту з попередньої лабораторної роботи застосовані стилі CSS, отримані знання та навички використання стилів.
